
# レベル

- 開始レベル: 基本的なセットアップとプロジェクト構造を理解する

- 中級レベル：自分でプロジェクトを一から作ることができる  
- プロレベル: 深い構造を理解し、複雑なプロジェクトを作成できる
   

# IT スキル

## Javascript(最もよく使う言語)

### ReactJS (中級レベル)
- Antd: コンポーネント
- Flutter CSS
- Axios: リクエストを処理する
- Firebase: レアルタイムデーターベース、ホステングとデプロイ。

### NextJS-TypeScript (開始レベル）

- App Routerを使用したファイルベースルーティング、Server/Client Componentsの使い分け、および基本的なデータフェッチの実装.

## Python (中級レベル)
- MLとDLモデルを使って、データーを纏める。

## Java (中級レベル)
- Object Oriented Programming

### Android (開始レベル)
- Android Studioを使って、アプリを作る。

### Spring (開始レベル)
- Mysqlに接続する
- 簡単なリクエストを処理する

## PHP (中級レベル)
- RESTful API: Reactなどのフロントエンドに適切なHTTPステータスコードでJSONデータを返すバックエンド、および外部APIからデータを取得してデータベースに保存する自動化ツール。

## Docker (開始レベル)

-Dockerfileの作成、イメージのビルド、コンテナの実行、およびDocker Composeを使用した複数コンテナ（PHPとDBなど）の構築。

## SQL, NoSQL (中級レベル)

- SQL および nosql でクエリを作成できる


## AWS (開始レベル)
- EC2、S3、RDS、Lambdaなどのコアサービスの基本操作、および基本的なクラウドインフラの構築。
